方案设计：
蓝牙点名系统的核心功能是通过蓝牙设备搜索周围用户，匹配数据库进行点名。
第一次需要将设备与同学身份进行匹配，所以要求同学将蓝牙名称改为自己的姓名，点名时会记录该设备的蓝牙地址，之后的点名就直接搜索蓝牙地址进行点名。
对于班级成员的录入，采用搜索excel文件（后缀为xls）进行录入。
主要功能：
1.导入excel文件读取里面内容存储数据库显示名单。
2.自动打开蓝牙设备进行搜索点名。
3.点名情况文件输出保存。
